# Assets Folder

This folder is where game art and sounds will live, starting around lesson 18.

## You don't need anything yet

For lessons 1–17, everything is drawn with simple colored squares and rectangles using Pygame's built-in drawing functions. No images or sounds required.

## When you do need assets (lessons 18+)

Starting in lesson 18 (Sprites & Animation) and lesson 19 (Sound & Music), you'll want some actual image and audio files. Here are great places to find free ones:

- **[OpenGameArt.org](https://opengameart.org/)** — huge library of free game art, sprites, and sounds
- **[Kenney.nl](https://kenney.nl/assets)** — high-quality free asset packs (the "1-Bit Pack" is perfect for a dungeon crawler)

## Suggested structure

```
assets/
├── sprites/
│   ├── player.png
│   ├── enemy.png
│   └── tiles/
│       ├── wall.png
│       ├── floor.png
│       └── door.png
└── sounds/
    ├── hit.wav
    ├── pickup.wav
    └── music.ogg
```

Until then, simple colored squares work perfectly fine. Don't worry about this folder yet — we'll set it up together when we get there.
