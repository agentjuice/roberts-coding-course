# 🐍 Robert's Coding Course

**Learn Python. Build Games. From zero to dungeon crawler.**

---

## What You'll Build

This course takes you from writing your first line of Python all the way to building a dungeon crawler game inspired by Minecraft Dungeons. Every lesson builds on the last, and every version is playable.

### 🎯 The Journey

| Project | Lessons | What You'll Learn |
|---------|---------|-------------------|
| **Warm-up** | 1-2 | Variables, loops, conditions, lists |
| **Connect 4** | 3-7 | 2D arrays, functions, Pygame, animation |
| **Snake** | 8-9 | Tuples, lists as data structures, classes |
| **Dungeon Crawler** | 10-20 | Tile maps, enemies, combat, loot, bosses, co-op |

### 🏗️ The Philosophy

- **Build real games, not toy examples.** Every lesson produces something you can play.
- **Start messy, then clean up.** You'll feel why functions and classes matter before you learn them.
- **No hand-waving.** You'll see the game loop, the state, the rendering — nothing is hidden.

## Setup

### 1. Install Python

- **Mac:** Open Terminal and run `brew install python` (or download from [python.org](https://python.org))
- **Windows:** Download from [python.org](https://python.org). Check "Add to PATH" during install!

Verify it works:
```bash
python3 --version
```

### 2. Install VS Code

Download from [code.visualstudio.com](https://code.visualstudio.com). This is your code editor.

Install the **Python extension** (search "Python" in the Extensions sidebar).

### 3. Install Pygame (needed from Lesson 5)

```bash
pip3 install pygame
```

### 4. Install numpy (needed for Connect 4)

```bash
pip3 install numpy
```

### 5. Your Workspace

Create a folder for the course:
```bash
mkdir ~/coding-course
cd ~/coding-course
```

Each lesson will go in its own file inside this folder. Keep everything — you'll want to look back at old code.

---

## How to Use This Course

1. **Read the lesson** on this site
2. **Type the code yourself** — don't copy-paste! Typing builds muscle memory.
3. **Run it** after each step to see what happens
4. **Do the experiments** — change things, break things, figure out why
5. **Try the challenge** before moving on

Take your time. There's no deadline. Some lessons will take 30 minutes, some might take a couple sessions. That's totally fine.
